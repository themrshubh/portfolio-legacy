AI-Based Virtual Assistant for Medical
Monitoring and Diagnosis
Back AWS Alexa Skill DynamoDB Sentiment Analysis
In this project, I aimed to create a system to aid psychiatrists in communicating better with their patients and staying up-to-date with the patients' progress. I designed numerous subsystems that handled different aspects of the project, like sentiment analysis, APIs, voice user interface, and data storage. I created multiple interfaces between these subsystems to enable robust communication between them to provide a connected and consistent system. I used Amazon's Alexa, which enabled users to record their thoughts, and it supported the psychiatrists to track patients' progress. The contextual nature of these conversations made them natural and intuitive. The custom-developed Alexa skill collected all the required parameters elegantly merely by engaging the user in the conversation. The use of sentiment analysis made the conversations more dynamic and gave Alexa a human touch. The system used highly scalable platforms to store user data, thus supporting upgradability. Using principles of Human-Machine interactions, we also developed a web application with a beautiful yet productive design. The web interface provided to the psychiatrists displayed user information on various levels of complexity. Thus, through this project, my team and I tried our best to deliver the functionalities promised while implementing experimental and innovative techniques.
